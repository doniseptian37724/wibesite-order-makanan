## Aplikasi Order Makanan

- Aplikasi ini adalah aplikasi order makanan
- Buat se sederhana mungkin
- aplikasi ini gabungan antara whatsapp dan microsite

## User Experience

1. User chat via whatsapp "Halo min, mau order"
2. Admin kirim link microsite
3. User pilih menu makanan di microsite, dengan meyebutkan jumlah porsi
4. Harga di set oleh admin microsite
5. Setelah user selesai order, user diarahkan kembali untuk chat via whatsapp dengan pesan "Halo min, saya sudah selesai order", di sertai detail orderan
6. Admin forward message pesanan ke merchant
7. Admin pickup pesanan ke merchant
